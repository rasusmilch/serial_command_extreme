NAME
  settings wifi set ssid - SSID

SYNOPSIS
  settings wifi set ssid <ssid> <password> <channel> <power> <enabled> <mode> <ratio>

DESCRIPTION
  WiFi set

ARGUMENTS
  ssid - SSID
  password - Pass
  channel - Channel
  power - Power
  enabled - Enable
  mode - Mode
  ratio

VALID VALUES
  ssid: string, 1..32 bytes
  password: secret, 8..64 bytes
  channel: unsigned integer, 1..11
  power: integer, -40..20
  enabled: on | off | true | false | 1 | 0
  mode: off | sta | ap
    off - Off
    sta - Sta
  ratio: decimal, -1.5..2.25

NOTES
  - Stored SSIDs are applied on the next WiFi reconnect.
  - Use quoted strings when the SSID contains spaces.

WARNINGS
  - Do not paste production passwords into shared logs.
  - Changing WiFi credentials can disconnect the console session.

EXAMPLES
  settings wifi set ssid "Shop AP" <new-password> 6 10 on sta 1.0
    Configure a WPA network using a placeholder password.
  settings wifi set ssid Lab <secret> 1 0 on ap 0.5
